## 题目描述

某校大门外长度为 L 的马路上有一排树，每两棵树的间隔为 1 米。我们可以把马路看成一个数轴，马路的一端在数轴 0 的位置，另一端在 L 的位置。马路上有一些区域要用来建地铁，这些区域用它们在数轴上的起始点和终止点表示。现在需要将树移走。请你计算移走后，马路上还有多少棵树。

## 解题思路

### 方法一：模拟数组

最直观的方法是使用一个数组模拟每米的情况。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int L, M;
    cin >> L >> M;

    vector<int> tree(L + 1, 1);

    for (int i = 0; i < M; i++) {
        int start, end;
        cin >> start >> end;
        for (int j = start; j <= end; j++) {
            tree[j] = 0;
        }
    }

    int count = 0;
    for (int i = 0; i <= L; i++) {
        count += tree[i];
    }

    cout << count << endl;
    return 0;
}
```

### 方法二：区间合并（更优）

使用差分数组优化，将区间操作优化到 O(1)。

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int L, M;
    cin >> L >> M;

    vector<int> diff(L + 2, 0);

    for (int i = 0; i < M; i++) {
        int l, r;
        cin >> l >> r;
        diff[l]--;
        diff[r + 1]++;
    }

    int count = 0, current = 0;
    for (int i = 0; i <= L; i++) {
        current += diff[i];
        if (current == 0) count++;
    }

    cout << count << endl;
    return 0;
}
```

## 复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 方法一 | O(L × M) | O(L) |
| 方法二 | O(L + M) | O(L) |

## 关键点

- 理解区间覆盖的概念
- 差分数组可以将区间修改优化到 O(1)
- 注意边界条件的处理
